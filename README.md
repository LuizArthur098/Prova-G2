## 🛠️ Tecnologias Utilizadas (Stack Obrigatória)
Front-end: React + Vite
Back-end: Node.js + Express
Banco de Dados: SQLite
Autenticação: JWT e bcrypt (Hash de senha)

## 🚀 Feature Extra
**Dark Mode:** Implementação dinâmica de troca de temas (Claro/Escuro) utilizando manipulação de classes CSS pelo React.

## ⚙️ Como Rodar a Aplicação

### 1. Rodando o Backend
Abra o terminal na pasta backend e execute:
\\\`bash
npm install
npm run dev
\\\`
O servidor rodará na porta 3000.

### 2. Rodando o Frontend
Em outro terminal, abra a pasta frontend e execute:
\\\`bash
npm install
npm run dev
\\\`
O React rodará no navegador (geralmente na porta 5173).
